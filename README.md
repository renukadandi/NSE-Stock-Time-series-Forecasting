Project Overview

This project is a comprehensive time-series forecasting solution developed to estimate future product sales patterns and predict NSE stock prices using Facebook Prophet. It integrates data cleaning, predictive modeling, and interactive visual analytics into a deployable web-based dashboard.

The system retrieves real-time stock market data through Yahoo Finance, produces forward-looking price forecasts along with uncertainty bands, and displays results via a user-friendly Streamlit application. The project is designed to support both analytical exploration and practical financial decision-making.

Key Highlights

Forecasting of time-series data using Facebook Prophet

Sales trend analysis based on historical datasets

Real-time NSE stock price retrieval using Yahoo Finance (yfinance)

Interactive and responsive dashboard built with Streamlit

Advanced visualizations powered by Plotly

Forecast confidence intervals for better risk assessment

Modular design allowing future integration of ARIMA, LSTM, or other models

Technology Stack

Language: Python

Libraries: Pandas, NumPy, Prophet

Visualization: Plotly, Matplotlib

Web Framework: Streamlit

Data Source: Yahoo Finance (yfinance)
